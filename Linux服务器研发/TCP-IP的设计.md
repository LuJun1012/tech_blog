# TCP/IP的设计

#### TCP/IP问题
* 滑动窗口如何实现
* sk_buff是什么
* TCP_NODELAY设置,针对的是对端收到包的时候的ACK发送回来是立即发送还是延迟发送，
* epoll检测网络IO的时候，ET和LT如何判断
* 服务器端出现大量close wait如何解决， 客户端大量close
* DDOS攻击 
 * 半连接队列越来越长，创建一些字节流结构体，导致内核协议栈数据不断变大，IO无法约束
 * 防火墙是等三次连接建立后才进行转发
 * 防火墙工作在ip层
 * DDOS攻击tcp层
 
* 发送和接收缓冲区设置为0会怎么样
* 如何发送syn包 raw socket
* buffer一般设置多大比较合适 流数据2MB, 字节数据KB字节
