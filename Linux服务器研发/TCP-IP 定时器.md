# TCP/IP 定时器
#### 种类
*  重传定时器
*  坚持定时器
*  保活定时器
*  2MSL定时器

#### 问题
* 多少个定时器
* tcp协议栈已经有保活机制，为什么在应用层还需要keeplive
* 重传的超时如何设置

#### 重传定时器
* 重传多少合适
* mtu=mss+tcphdr + iphdr

####  保活定时器
* 内核协议栈的keeplive回主动释放连接
* 应用层做一层心跳包

#### timewait
* 保证对端释放掉fd
* 保证发起方的ACK必须到达对端

#### 单位
* cwind， ssthresh, mss 单位统一
