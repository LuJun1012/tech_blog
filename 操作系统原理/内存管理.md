# 内存管理
#### 管理角度
* 第一，物理内存的管理，相当于会议室管理员管理会议室。

* 第二，虚拟地址的管理，也即在项目组的视角，会议室的虚拟地址应该如何组织。

* 第三，虚拟地址和物理地址如何映射，也即会议室管理员如果管理映射表

#### 内存栈空间
* Text Segment
* Data Segment
* BSS Segment
